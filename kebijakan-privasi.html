Kebijakan Privasi

Pembaruan Terakhir: 14 Juni 2026

Selamat datang di Ledgerly. Kami sangat menghargai kepercayaan Anda dan berkomitmen penuh untuk melindungi privasi serta keamanan informasi bisnis dan data pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda saat menggunakan platform kami.

1. Informasi yang Kami Kumpulkan

Untuk mendukung operasional pencatatan keuangan dan pengelolaan inventaris Anda, kami mengumpulkan informasi sebagai berikut:

  • Informasi Pendaftaran Akun: Nama lengkap pemilik, nama bisnis/UMKM, alamat email aktif, kata sandi terenkripsi, dan nomor telepon/WhatsApp.
  • Data Operasional Bisnis: Informasi produk (nama barang, kategori, stok, harga beli, harga jual, barcode, dan nama/WhatsApp supplier) yang Anda simpan di dalam katalog inventaris.
  • Data Transaksi: Catatan transaksi masuk (restock/pembelian) dan transaksi keluar (penjualan), termasuk jumlah unit, total nominal, tanggal transaksi, metode pembayaran, dan catatan transaksi.
  • Riwayat Chatbot AI: Percakapan atau kueri obrolan Anda dengan asisten Ledgerly AI yang disimpan secara lokal di peramban Anda untuk pengalaman asisten yang dipersonalisasi.

2. Penggunaan Informasi

Informasi yang kami kumpulkan digunakan secara eksklusif untuk kepentingan berikut:

  • Menyediakan layanan dashboard inventaris dan keuangan real-time untuk mempermudah operasional usaha Anda.
  • Memproses data input mutasi stok secara otomatis melalui modul analisis chatbot AI kami.
  • Mengirimkan notifikasi peringatan stok rendah (*reorder point alert*) secara otomatis ke nomor WhatsApp pemilik toko.
  • Menghasilkan analisis data, seperti produk terlaris dan rekomendasi jumlah restock produk.
  • Memungkinkan ekspor laporan berkas ke dalam format PDF, CSV, atau Excel (XLSX).

3. Keamanan Data & enkripsi

Kami merancang sistem dengan standar keamanan modern guna melindungi data Anda dari akses tanpa izin:

  • Row-Level Security (RLS): Basis data kami pada cloud Supabase diproteksi secara ketat menggunakan kebijakan RLS. Setiap data produk, transaksi, dan kategori diisolasi berdasarkan identitas pengguna (`auth.uid()`), sehingga tidak ada satu pun UMKM yang dapat membaca atau memodifikasi data milik UMKM lain.
  • Enkripsi Transportasi Data: Semua lalu lintas data antara browser Anda dengan serverless functions dan database dienkripsi menggunakan protokol HTTPS yang aman (TLS).
  • Rahasia Kunci API: Kunci integrasi eksternal (seperti Gemini AI dan WhatsApp Gateway) disimpan secara rahasia di sisi server (*serverless environment*) dan tidak pernah diekspos ke publik atau dideklarasikan di browser klien.

4. Berbagi Informasi dengan Pihak Ketiga

Ledgerly tidak pernah menjual, menyewakan, atau membagikan data transaksi maupun inventaris bisnis Anda kepada pihak ketiga untuk kepentingan iklan. Kami hanya terintegrasi dengan layanan pihak ketiga yang terpercaya guna menunjang fitur aplikasi:

  • Supabase Inc. sebagai penyedia infrastruktur cloud database PostgreSQL dan manajemen otentikasi.
  • Google Gemini AI API sebagai pemroses kecerdasan buatan (*natural language processing*) pada asisten chatbot Ledgerly AI.
  • WhatsApp Gateway API untuk pengiriman pesan notifikasi stok rendah.

5. Kontrol dan Hak Pengguna

Sebagai pemilik data, Anda memiliki kontrol penuh atas informasi Anda di Ledgerly:

  • Anda dapat mengubah, menambah, atau menghapus data katalog produk dan riwayat transaksi kapan saja langsung dari dashboard.
  • Anda dapat menghapus riwayat obrolan dengan asisten AI secara instan dari panel chat.
  • Jika Anda ingin menghapus akun dan seluruh data operasional bisnis Anda secara permanen dari server cloud kami, Anda dapat menghubungi kami melalui administrator.

6. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu untuk menyesuaikan perkembangan fitur Ledgerly. Kami menyarankan Anda untuk meninjau halaman ini secara berkala.

Hubungi Kami

Jika Anda memiliki pertanyaan mengenai Kebijakan Privasi ini atau masalah keamanan data Anda, silakan hubungi tim pengembang Ledgerly TP-G007 melalui email di support@ledgerly.my.id.
